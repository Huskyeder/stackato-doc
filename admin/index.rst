Stackato Admin Guide
====================

.. only:: not public

  .. note:: **internal**:
     You are viewing the Stackato *internal* documentation. To edit the
     documentation, clone `stackato-doc.git
     <https://github.com/ActiveState/stackato-doc>`_, modify the `*.rst`
     files and then follow *README.md* to publish the changes.
     

`Stackato <http://www.activestate.com/cloud>`_ is ActiveState's cloud
application framework for creating a multi-language, secure and
private Platform-as-a-Service (PaaS). This guide covers how to set it
up and how to configure it for your end users.
